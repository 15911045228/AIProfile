# 拼豆店客服知识库

## 目录结构

```
客服知识库/
├── 01_店铺介绍.md          # 店铺定位、人群、特点
├── 02_价格优惠.md          # 标准价格 + 当前活动 + 历史活动
├── 03_营业时间.md          # 10:00-20:00
├── 04_预约流程.md          # 引导话术（几位+哪天+几点）
├── 05_到店路线.md          # 地铁/停车/地标
├── 06_新手常见问题.md       # 不会做/多久/年龄/价格
├── 07_作品制作问题.md       # 颜色不够/熨烫/损坏
├── 08_售后处理.md          # 熨坏/不满意处理流程
├── 09_客服回复规范.md       # 语气/长度/禁忌词
├── 10_营销引导.md          # 嫌贵/考虑中/先看看
├── 11_敏感问题处理.md       # 比价/退款/差评
├── 12_知识更新记录.md       # AI规则，非话术
├── 13_客户意图库.md         # 意图识别+策略映射
├── 14_优秀对话案例.md       # 学习样本（模板待填充）
└── 15_客服复盘库.md         # 改进追踪（模板待填充）
```

## 三层结构

1. 基础信息层（01-05）：店铺事实
2. 问答服务层（06-11）：FAQ + 话术 + 敏感问题
3. 运营增强层（12-15）：意图识别 + 案例沉淀 + 复盘改进

## 关键设计

- 价格和活动合并（02），客户问价时一并回答
- 历史活动单独区段，结束的活动移入
- 知识更新记录标注为 AI 规则（非话术），避免被引用回复
- 意图库（13）让 AI 识别意图后选择最优回复策略
