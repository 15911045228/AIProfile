# 个人 Profile 总结

## 核心身份

AI 应用开发方向学习者，目前从事新能源行业（投标、需求分析、技术方案），同时经营拼豆店。

## 长期目标

### P0：成为 AI 应用工程师
- Agent 开发 / MCP 开发 / Prompt Engineering / Workflow 设计
- Tool Calling / RAG / AI 自动化 / AI 产品设计 / Vibe Coding
- AI 项目架构设计

### P1：积累具面试竞争力的 AI 项目
- 真实可落地、能解决业务问题、有工程价值
- 可写进简历、面试能讲清设计思路

### P2：AI 提高工作效率
- 投标文件、技术方案、Word/Excel/PPT 自动生成
- 需求分析、信息提取、文档自动生成

### P3：AI 辅助拼豆店运营
- AI 客服、内容创作、短视频、获客、私域运营

## 学习风格

- 原理 → 设计思路 → 能力边界 → 适用场景 → 实践
- 偏好多种方案对比，工程思维
- 重视面试竞争力

完整流程：需求分析 → 产品设计 → 流程设计 → Agent/Tool/MCP/Prompt 设计 → MVP → 测试 → 优化 → 沉淀文档
