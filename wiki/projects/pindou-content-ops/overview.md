# 拼豆店内容运营追踪

## 概述

拼豆店短视频内容运营的数据追踪系统，记录每日选题、发布效果，通过数据驱动优化内容策略。

关联个人目标：P3 - AI 辅助拼豆店运营（内容创作、短视频、获客）

## 追踪指标

| 指标 | 说明 |
|------|------|
| 播放量 | 视频发布后 7 日累计播放 |
| 点赞率 | 点赞 / 播放量 |
| 收藏率 | 收藏 / 播放量 |
| 评论数 | 用户评论数量 |
| 转发率 | 转发 / 播放量 |
| 涨粉 | 发布后 7 日新增粉丝 |
| 完播率 | 完整播放率（如有数据）|
