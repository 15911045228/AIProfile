# 分析与优化建议

定期对选题效果数据进行复盘，提炼规律并优化策略。

## 分析方法

1. **横向对比**：同一时段各选题的播放量、互动率对比
2. **纵向趋势**：账号整体数据随时间的变化趋势
3. **归因分析**：爆款/扑街原因推断（选题、封面、BGM、发布时间、平台差异）
4. **AB 测试**：同类选题不同呈现方式的对比

## 优化方向

| 维度 | 可优化项 |
|------|----------|
| 选题 | 什么类型的内容播放量最高？ |
| 标题/封面 | 什么标题模板点击率更高？ |
| BGM | 什么风格音乐完播率更好？ |
| 发布时间 | 什么时段发布互动率最高？ |
| 平台 | 同样内容在小红书 vs 抖音表现差异？ |
